## █░░ WHAT IS THIS

**TYPOCHE** is a minimal, brutalist-themed typing speed test built with vanilla HTML, CSS, and JavaScript. No frameworks. No bloat. Just raw keystrokes and cold, hard stats.

> Fast feedback. Dark aesthetic. Monospace everything.

---

## █░░ FEATURES

```
[✓] Timed modes     — 15s / 30s / 60s / 120s
[✓] Word modes      — 10w / 25w / 50w
[✓] Free mode       — type without limits
[✓] Live WPM & RAW  — real-time stats as you type
[✓] Progress bar    — visual test progress tracking
[✓] Test history    — LOG panel with past results
[✓] Theme toggle    — DARK / LIGHT
[✓] Noise canvas    — CRT-style background texture
[✓] Quote system    — random quotes with author credit
[✓] Keyboard nav    — tab to restart, esc for new quote
```

## █░░ GETTING STARTED

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/typoche.git

# Navigate into it
cd typoche

# Open in browser (no build step needed)
open index.html
```

> No dependencies. No `npm install`. Just open and type.

## █░░ KEYBOARD SHORTCUTS

| Key | Action |
|-----|--------|
| `Tab` | Restart current test |
| `Esc` | Load a new quote |
| `Click` | Focus the typing area |

---

## █░░ MODES

| Mode | Description |
|------|-------------|
| `15s` `30s` `60s` `120s` | Race the clock |
| `10w` `25w` `50w` | Fixed word count |
| `free` | No timer, no limit |

---

## █░░ STATS TRACKED

- **WPM** — corrected words per minute
- **RAW** — unfiltered keystrokes per minute
- **Accuracy** — correct characters vs total typed
- **Time** — duration of the test
